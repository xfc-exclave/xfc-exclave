# EasyMark - 让写作更简单！

EasyMark 是一款所见即所得的 Markdown 编辑器。

🍼 ~~ 你可以前往我们的～～github 主页下载安装程序进行体验。

EasyMark 既支持 commonmark 语法，也支持 GFM 语法。

您可以点击查看源码按钮，或使用快捷键 Ctrl + / 来查看 markdown 源文本。

  • 特性

  • 开发者: Chinmoku

你可以添加行内代码例如 inline code 和代码块：

Tips: 使用 Mod-Enter 来退出块级元素，例如代码块。

你可以输入 || 和一个空格来创建表格：

表头 1表头 2
内容 1内容 1
内容 2内容 2

数学公式通过 TeX 表达式支持。

这里我们有行内公式： E=mc2E = mc^2E=mc2，你可以点击并编辑它。

数学公式块也是支持的。

你可以输入 $$ 和一个空格来创建数学公式块。

使用 emoji 快捷指令 例如 :+1: 来添加 emoji.

在输入时，你也许注意到了 emoji 过滤器，尝试输入 :baby 来查看它。
